# T-Rex

灵巧手触觉操作数据（LeRobot v3.0）。

- **训练入口**: `train/trex_dataset`
- **注册表**: `manifests/mix.yaml` → `t_rex`
- **下载**: `bash datasets/t_rex/scripts/download.sh`
- **监控**: `bash datasets/t_rex/scripts/watch_download.sh`

HF: [zekaiwang/trex_dataset](https://huggingface.co/datasets/zekaiwang/trex_dataset)
